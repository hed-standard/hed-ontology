HED ontology project
--------------------

The HED ontology project is an effort to map HED schemas, which are hierarchical
vocabularies designed for annotation, into formal ontologies.
The goal is to provide additional metadata that is not accommodated in the
HED schema files.

HED is an open research community effort; others interested are invited to participate and contribute.
Visit the `HED project homepage <https://github.com/hed-standard>`_ for links to the latest developments..
